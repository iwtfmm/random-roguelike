# 集成战略 · 随机开局终端

基于明日方舟集成战略（IS）数据的随机开局抽签器。一键随机抽取 主题 / 难度 / 分队 / 招募组合 / 结局链，并支持自限模块随机禁用干员职业。

在线体验：<https://random-roguelike.vercel.app/>

## 功能

### 主抽签模块

- **一键随机**：依次揭晓 主题 → 难度 → 分队 → 招募组合 → 结局链
- **主题池**：可固定多个主题，仅从中随机
- **元素开关**：主题 / 难度 / 分队 / 招募任选参与随机，关闭的元素使用固定值
- **每主题池配置**：限定难度上下限、勾选参与随机的分队、招募组合与结局
- **随机结局**：每主题含 5 个结局，遵循互斥规则与连打链路（1/2 互斥为前置，3/4 互斥为后继，5 独立）
- **历史记录**：最近 20 次方案回看

### 自限模块

- **随机禁用职业**：从 8 大干员职业中随机禁用 1~n 个，独立配置、独立抽取
- **禁用池与数量**：可勾选允许被禁用的职业子集，并设置最少/最多禁用数量

## 技术栈

React 18 + TypeScript + Vite + Tailwind CSS + Zustand + lucide-react

## 本地开发

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:5173/

## 构建

```bash
npm run build
```

产物在 `dist/`，自包含（相对路径），双击 `index.html` 即可打开。

## Windows 快捷启动

双击 `startup.bat` 或桌面快捷方式「随机肉鸽抽签器」即可启动并自动打开浏览器（需先 `npm install`）。
